# Java Guessing Game

This project is an interactive Java application that generates a random number from 1 to 10 and allows the user to guess the correct number. The program provides feedback for each guess and tracks the number of attempts required to find the correct answer.

## Features

- Random number generation
- User input using JOptionPane
- Input validation
- Attempt tracking
- Feedback for high and low guesses
- Personalized success messages

## Skills Demonstrated

- Java
- Loops
- Conditional Logic
- User Input
- Random Number Generation
- JOptionPane Dialogs
